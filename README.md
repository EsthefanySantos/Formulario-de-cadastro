## 💻 Sobre o projeto

Este projeto é uma interface de tela de cadastro desenvolvida utilizando HTML e CSS, com foco em organização visual, responsividade e boas práticas de layout.

O objetivo foi praticar e aprimorar habilidades de desenvolvimento front-end, criando uma tela moderna e intuitiva para cadastro de usuários.

<!-- MODELO FUNCIONALIDADES: -->
## ⚙️ Funcionalidades
- [x] Formulário de cadastro com:
- [x] Nome e sobrenome
- [x] Email
- [x] Celular
- [x] Senha e confirmação de senha
- [x] Seleção de gênero
- [x] Layout dividido (ilustração + formulário)
- [x] Design limpo e moderno
- [x] Botão de envio estilizado
